Biometric Data Processing

This system processes facial biometric data. Ensure you have:

  • Obtained explicit consent from all registered users
  • Informed users about data collection and storage
  • Implemented data retention policies

Data Storage

All data is stored locally. No external cloud services are used. Face embeddings (128-dimensional vectors) are stored instead of raw facial images where possible.
